# Projeto: Análise de Cancelamento de Clientes

Este projeto tem como objetivo entender os principais fatores que levam clientes a cancelar um serviço, a partir de uma base de dados com informações de comportamento, contratos, ligações ao call center e atrasos de pagamento. A análise é feita com Python, utilizando as bibliotecas pandas e plotly para tratamento de dados e visualização.

## Como rodar o projeto

Instale o Python e as bibliotecas necessárias:

pandas
plotly
jupyter (opcional, para abrir o notebook)

Faça a instalação com o comando:

`pip install pandas plotly jupyter`

Coloque o arquivo de dados na mesma pasta do notebook.

Abra o arquivo inicial.ipynb em um ambiente como Jupyter Notebook ou Visual Studio Code e execute as células em ordem.

## Etapas do projeto

1. Importação da base de dados.
2. Exploração inicial da tabela e limpeza dos dados.
3. Análise da taxa de cancelamento.
4. Criação de gráficos para identificar padrões por coluna.
5. Aplicação de filtros e avaliação do impacto das condições mais relevantes no cancelamento.
6. Resultados esperados

## A análise permite identificar os principais motivos associados ao cancelamento, como:

contratos mensais;
grande quantidade de ligações ao call center;
atrasos frequentes no pagamento.

Com base nesses insights, é possível sugerir ações para reduzir o cancelamento, como melhorias no atendimento, alertas para clientes em atraso e incentivos para contratos mais longos.

Após a análise, foi concluído que se os três problemas forem resolvidos, a taxa de cancelamentos de cliente será reduzida de 56% para 18%, aproximadamente.

Este projeto é uma introdução prática à análise de dados com Python e pode servir como base para decisões mais assertivas de negócio.
